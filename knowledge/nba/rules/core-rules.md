# Game Structure

NBA 正式比赛由四节组成，每节十二分钟。常规时间结束后如果比分相同，会进入五分钟加时赛，直到分出胜负。比赛节奏之所以比很多业余联赛更快，一方面是四节制和暂停管理更精细，另一方面是进攻时间、回场、发球和换人规则都强调连续性。

NBA 的主教练和球员管理比赛的方式，通常会围绕回合价值展开。因为一场比赛的回合数有限，所以每一次失误、每一次无效犯规、每一次低质量出手都会放大影响。理解 NBA 规则时，最好不要把它只看成“裁判怎么吹”，而要同时理解这些规则为什么会塑造现代比赛的节奏。

# Shot Clock and Backcourt

NBA 进攻方通常拥有二十四秒完成一次有效进攻。有效进攻不是单纯把球投出去，而是要在时间结束前出手，并且篮球触及篮筐。如果球出手后碰到篮筐并且进攻方继续控制球，通常会获得新的进攻时间；不同回合情境下，重置后的时长并不完全相同。

后场推进也有严格限制。球队在后场获得控球后，必须在规定时间内把球推进到前场。如果球已经合法进入前场，再把球主动带回后场，会构成回场违例。这个规则的核心目的，是防止球队拖慢节奏、无限消耗时间。

# Violations and Fouls

违例和犯规的区别很重要。违例通常指的是非法动作或时间、位置类错误，例如走步、两次运球、脚踢球、三秒违例、回场违例。犯规则是对对手身体接触、非法防守或比赛秩序的破坏。违例一般导致交换球权，犯规则可能导致边线发球、罚球，或者累计到全队犯规数后带来额外处罚。

防守三秒是 NBA 比较有代表性的规则。它要求防守球员不能长时间站在禁区内而又不积极防守对位对象。这个规则的存在，是为了给突破和持球进攻创造空间，避免禁区里长期站着协防中锋破坏观赏性。

# Goaltending and Basket Interference

干扰球和篮筐干扰是很多新球迷容易混淆的概念。简单理解，干扰球更多发生在球的飞行阶段，当球已经在下降轨迹并且有机会入筐时，防守方不能非法触碰。篮筐干扰则更强调对篮筐、篮板和篮网周围空间的非法接触，尤其是在球仍然有可能通过篮筐圆柱体时。

这些规则的目标，是保护一次合法投篮的自然结果。如果没有这些限制，球员可以在球接近篮筐时直接把球拨走，投篮命中率和战术价值都会被破坏。
