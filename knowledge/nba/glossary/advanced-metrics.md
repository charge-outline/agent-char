# True Shooting Percentage

True Shooting Percentage，常写成 TS%，用来衡量球员得分效率。它比普通命中率更全面，因为它同时把两分球、三分球和罚球都考虑进来了。一个球员如果三分和罚球都很强，TS% 往往会比只看投篮命中率更能反映他的真实效率。

TS% 高，通常意味着球员每次出手能换回更多分数。但它并不直接等于进攻能力强于一切，因为它没有单独表示持球压力、失误代价、战术牵制和关键球难度。真正分析球员时，TS% 最适合与使用率、助攻失误比和回合占有情况一起看。

# Effective Field Goal Percentage

eFG% 是对普通投篮命中率的修正，因为三分球的价值高于两分球。它会给命中的三分球更高权重，从而更准确地描述投篮结构带来的收益。一个大量命中三分的球员，eFG% 可能会显著高于传统命中率。

eFG% 特别适合比较两名不同出手结构的球员。例如，一个中距离占比很高的球员，即便命中率不低，也可能因为缺少三分和罚球而在 eFG% 上落后于空间型后卫。

# Usage Rate

Usage Rate 通常用来描述球员在场时，有多少回合以他的出手、失误或制造罚球结束。它不是触球率，也不是持球时间。一个球员可以经常持球组织，但如果他把回合更多地分配给队友，Usage Rate 不一定很高。

高使用率往往意味着更大的战术责任。球队把更多决定权交给这个球员，他需要处理更多强防守回合、关键时间球和临时应变。高使用率与高效率同时存在时，通常说明球员具备很强的进攻核心价值。

# Offensive and Defensive Rating

Offensive Rating 通常描述球员或球队每一百回合大致能得到多少分，Defensive Rating 则描述每一百回合大致会让对手得到多少分。它们本质上是节奏修正后的效率指标，适合跨球队、跨阵容、跨节奏环境做比较。

不过，单看个人防守效率时要特别小心，因为它会强烈受队友、轮换体系和对位策略影响。一个球员所在阵容的防守沟通很好，他的个人 Defensive Rating 可能就会比实际单防能力显得更漂亮。
